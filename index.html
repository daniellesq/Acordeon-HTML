Cheat Sheet HTML

< !DOCTYPE html >

Una declaración al inicio de html que le indica al navegador que version de HTML se está usando, en este caso HTML5.

< Html >

Es el contenedor principal de toda la página web, dentro se encuentran las secciones head y body

< Title >

Define el nombre de la página, es el texto que aparece en la pestaña del navegador

< Head >

Contiene información sobre la página web, incluye metadatos, enlaces a hojas de estilo, scripts, entre otros

< Body >

Contiene el contenido visible de la página: texto, imagenes, formularios, entre otros

< P >

se usa para representar párrafos de texto

Secciones semanticas

  • Header: Encabezado de la página.

  • Nav: Menú de navegación.

  • Main: Contenido principal.

  • Aside: Barra lateral o información adicional.

  • Footer: Pie de página con información.

< h1 > - < h6 >

Encabezados con jerarquia: h1 siendo el de mayor tamaño y h6 el de menor

< Div >

Div: Contenedor en bloque, usado para agrupar secciones completas

< Span >

Span: Contenedor en linea, usado para marcar texto

Listas

  • OL: lista ordenada (numerada)

  • UL: Lista no ordenada (con viñetas)

  • LI: Elemento dentro de una lista

  • DL: Lista de definiciones

  • DT: Término de la definición

  • DD: Descripción del término

Tablas

  • Table: Crea una tabla

  • TR: Fila de tabla

  • TD: Celda de datos

  • TH: Celda de encabezado

  • Thead: Encabezado de la tabla

  • Tbody: Cuerpo de la tabla

  • Tfoot: Pie de la tabla

Multimedia

  • IMG: Inserta una imagen, requiere su ruta y un texto alternativo

  • Audio: Reproduce archivos de sonido

  • Video: Reproduce archivos de video

  • Source: Define la fuente de un audio o video

Formularios

  • Form: Agrupa todos los campos del formulario

  • Input: Campo de entrada: texto, numero, email, contraseña, etc.

  • Select: Lista desplegable

  • Option: Opcion dentro de un select

  • Textarea: Area de texto multilinea

  • Label: Texto descriptivo asociado a un campo de formulario.
